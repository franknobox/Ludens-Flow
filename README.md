# Ludens-Flow
This is a project warehouse established to participate in the 2026 SUAT AI Agent Innovation Competition.

**Ludens-Flow** is a multi-agent, artifact-centric workflow system for game development.

The name combines:
- **Ludens**, from *Homo Ludens*, emphasizing games as a creative human process.
- **Flow**, referring to a structured, controllable development pipeline.

Ludens-Flow does **not** aim to automatically generate complete games.
Instead, it focuses on **structuring, coordinating, and reviewing the game development process itself**.

By orchestrating multiple specialized agents around explicit artifacts
(e.g. GDD, project plans, implementation paths, and reviews),
Ludens-Flow makes development decisions **traceable, reviewable, and reproducible**.

**Ludens-Flow** 是一个多智能体、以工件为中心的游戏开发工作流程系统。

名称结合了以下内容：
- **Ludens**，源自*Homo Ludens*，强调游戏作为一种创造性的人类过程。
- **Flow**，指结构化、可控的开发流程。

Ludens-Flow**并不**旨在自动生成完整游戏。
相反，它专注于**结构化、协调和审查游戏开发过程本身**。

通过围绕显式伪造物协调多个专门代理
（例如GDD、项目计划、实施路径和评审），
