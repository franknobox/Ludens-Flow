# 铁魂 · Iron Equilibrium  
**GDD v0.1** —— 可落盘版本

---

## 1. 概述
- **一句话卖点**  
  一款以“负重-平衡”为核心的慢节奏类魂动作游戏——每一次挥剑都在与自身重量博弈，被打碎平衡即面临致命硬直。

- **目标体验**  
  1. 铠甲越重，安全感越强，却越像镣铐。  
  2. 战斗像打铁：火星四溅、节奏沉重，胜负取决于“拆招”而非“翻滚”。  
  3. 玩家始终处于“再穿一件就超重”的临界焦虑，装备选择=风险调校。

---

## 2. 核心循环（分钟级）
```
探索 → 负重决策 → 遭遇敌人 → 攻防拆招 → 平衡值博弈 → 击杀/硬直惩罚 → 资源掉落 → 回篝火重配装备
```

| 节点 | 关键问题 | 系统支撑 |
|------|----------|----------|
| 负重决策 | 我要不要换这件重甲？ | 动态负重→平衡恢复曲线实时可视化 |
| 拆招 | 我该格挡、弹反还是轻击打断？ | 武器冲击等级 vs 护甲稳定值 |
| 硬直惩罚 | 被打崩平衡后我能否活命？ | 长硬直+敌人处决动画，队友可援护（联机） |

## 3. 关键系统

### 3.1 动态负重 & 平衡值（Balance, BAV）
- **公式**  
  `BAV_max = 80 - (装备重量 - 阈值1) / 衰减系数`  
  `BAV_recovery = base × 护甲重量惩罚% × 站立/格挡/攻击状态系数`

- **核心规则**  
  1. 任何动作（走/攻/格挡/受击）扣BAV。  
  2. BAV≤0 → 进入“失衡”状态：  
     - 硬直2.5s，期间受到+50%伤害，可被处决。  
     - 恢复后BAV回满，但进入30s的“脆弱”debuff：BAV_max-20%。  
  3. 超重阈值可临时突破：喝“铁骨精油”+15%阈值，持续90s，结束后立刻-30% BAV_max作为反噬。

### 3.2 武器冲击等级（Impact Level, IL）
- 每把武器/技能附带IL 1-5。  
- 格挡方稳定值（Stability, STB）≥ IL → 成功拆招，不扣BAV并反弹对手BAV。  
- STB < IL → 扣BAV，出现“踉跄”小硬直。  
- 双手握持+1 IL，跳劈+1 IL，蓄力+2 IL。

### 3.3 装备部位 & 局部防护
- 头/胸/手/腿四部位，独立重量与防护值。  
- 局部破坏：敌人可对部位累积“破损值”，满后该部位防御-50%且额外+20%重量（甲片下垂）。  
- 维修：只能在篝火消耗“铸铁碎片”，碎片=主要货币。

### 3.4 篝火 & 重配
- 篝火=存档+升级+装备重配。  
- 重配无成本，但“铸铁碎片”随身携带死亡即全掉落（魂系传统）。  
- 新增“负重校准”面板：可存3套配装方案，一键切换。

### 3.5 联机（轻量级）
- 召唤印记：最多1盟友+1敌对入侵。  
- 盟友可执行“援护打击”：在宿主失衡瞬间攻击敌人，成功则取消宿主处决动画。  
- 入侵者获得“轻量化”buff：装备重量-30%，但BAV_max-25%，鼓励速攻。

## 4. 关卡结构（宏观）

| 区域 | 主题 | 负重玩法钩子 | Boss机制 |
|------|------|--------------|----------|
| 1. 铸铁堡垒（教学） | 狭窄栈道、熔岩喷口 | 超重会压塌木板 | 熔炉骑士：IL高，教玩家拆招 |
| 2. 悬空铁链谷 | 仅30cm宽铁链 | 超重=风吹失衡掉落 | 无Boss，环境即敌人 |
| 3. 齿轮图书馆 | 可升降书架 | 齿轮机关定时改变地面高度，重甲难跳跃 | 档案守护者：切换IL模式，需实时换装备 |
| 4. 黑水矿坑 | 淤泥减速+50%重量感 | 越重越陷，需轻甲滑冰 | 淤泥巨像：吸收玩家掉落铸铁碎片成长 |
| 5. 终焉铁穹 | 零重力环→重量失效 | 传统负重规则反转：越轻越难控制惯性 | 铁魂本体：双阶段，先重甲后脱壳极速 |

**流程长度**：主线≈25小时，全收集+支线≈40小时。  
**关卡连通**：中后期获得“滑轮钩”可快速升降，形成立体捷径。

## 5. 美术风格
- **关键词**：锈蚀、铸铁、笨重、低饱和、火星高光。  
- **色板**：主灰+暗银，点缀熔岩橙与机油绿。  
- **材质**：  
  - 铠甲：高模铸造痕+法线凹坑，Shader带可变锈蚀度（随玩家死亡次数增加）。  
  - 武器：锋口持续发光，IL越高越亮。  
- **角色比例**：头身比1:8.5，肩宽夸张，强调“人形坦克”。  
- **UI**：  
  - BAV槽=铁水烧红玻璃管，空管时裂纹碎裂。  
  - 装备界面直接显示“预计硬直时长”数值，减少信息焦虑。

## 6. 不做什么（边界）
1. 不做高速翻滚/闪避类魂（取消无敌帧翻滚）。  
2. 不做“负重仅影响移动速度”的浅层设计。  
3. 不做随机数值掉落，所有装备均为固定位置或固定精英掉落。  
4. 不做开放世界，保持类魂式互联箱庭。  
5. 不做多结局叙事，仅单线+隐藏后日谈，控制脚本成本。  
6. 不做内购/赛季通行证，DLC仅扩展新区域与装备。
